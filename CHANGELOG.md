# Change Log

## 2.0.0

**Added**

- Add support for PHP `~8.0.12` and `~8.1.0`.
- Drop support for PHP `7`.
- Compatibility with Laravel Lumen framework.

**Changed**

- Runner will now trigger warning instead of error when able to generate tests for a class.
